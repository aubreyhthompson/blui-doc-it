# Visualizations

<ImageGrid regularWidth images={[VisualizationBanner]} />

There are many opportunities to visualize data in different applications. The most common visualizations used in PX Blue applications come in the form of charts / graphs and maps.

<Divider />

# Charts / Graphs

<ImageGrid
    images={[LineChart, PieChart]}

There are [many variations](https://material.io/design/communication/data-visualization.html#types) of data visualizations. They can be a big line chart with 8 series, occupying the entire screen; or they can be as tiny as a simple pie chart embedded in [lists](/patterns/lists).

# Developers

There are many libraries and services for data visualizations. PX Blue currently provides themes for the following libraries:

**Angular**, **React** and **Ionic**:

-   Charts & Graphs

    -   [Highcharts](http://www.highcharts.com) ([Pricing](https://shop.highsoft.com/))
    -   [@pxblue/highcharts](https://www.npmjs.com/package/@pxblue/highcharts)
    -   [ChartJS](https://www.chartjs.org/) (Open Source)
    -   [@pxblue/chartjs](https://www.npmjs.com/package/@pxblue/chartjs)

-   Maps
    -   [Mapbox](https://www.mapbox.com/) ([Pricing](https://www.mapbox.com/pricing/))
    -   [@pxblue/mapbox](https://www.npmjs.com/package/@pxblue/mapbox)
