### Px Blue Component Libraries

The all new [PX Blue Components](https://pxblue-components.github.io/) library page is now available and is a new addition to the existing PX Blue documentation
thats available through the PX Blue documentation site or directly from the link above.

The component libraries have each been re-designed and have an all new experience with screen shots, API documentation and interactions.

We have been continuing to develop new and update exising components to build out each component library.

-   For React, we have added components for [ChannelValue](https://pxblue-components.github.io/react/?path=/info/components-channel-value--get-read-me-story), [Drawer](https://pxblue-components.github.io/react/?path=/info/components-drawer--get-read-me-story),
    [EmptyState](https://pxblue-components.github.io/react/?path=/info/components-empty-state--get-read-me-story), [Hero](https://pxblue-components.github.io/react/?path=/info/components-hero--get-read-me-story),
    [ListItemTag](https://pxblue-components.github.io/react/?path=/info/components-list-item-tag--get-read-me-story), [Spacer](https://pxblue-components.github.io/react/?path=/info/components-spacer--get-read-me-story) and
    [UserMenu](https://pxblue-components.github.io/react/?path=/info/components-user-menu--get-read-me-story)

    -   Also several React components updates for [InfoListItem](https://pxblue-components.github.io/react/?path=/info/components-info-list-item--get-read-me-story) and [ScoreCard](https://pxblue-components.github.io/react/?path=/info/components-scorecard--get-read-me-story).

-   For React Native, we added components for [ChannelValue](https://pxblue-components.github.io/react-native/?path=/info/api-documentation--channel-value), [EmptyState](https://pxblue-components.github.io/react-native/?path=/info/api-documentation--empty-state),

### PX Blue NPM Packages now available in TypeScript

To take advantage of it's many benefits, new resources for React and React Native will be built using Typescript instead of plain JavaScript. We have also begun the process of migrating our existing resources to TypeScript.
