# PX Blue CLI

<EmbeddedYouTubePlayer videoID={`CESJ9BKLWco`} />

The PX Blue CLI (Command-Line Interface) is a tool for quickly starting new projects using PX Blue. It will create a skeleton project in your chosen framework and automatically integrate:

-   Material and PX Blue component libraries
-   PX Blue themes
-   Open Sans Typography package
-   PX Blue colors package
-   PX Blue code formatting and linting profiles (optional)

When starting a new project, you will have the opportunity to select a template as your starting point, including a blank template, basic routing template, and a full authentication template.

> These templates are available for [Angular](https://github.com/pxblue/angular-cli-templates/blob/master/README.md#angular-cli-templates), [React](https://github.com/pxblue/react-cli-templates/blob/master/README.md#react-cli-templates), [React-Native](https://github.com/pxblue/react-native-cli-templates/blob/master/README.md#react-native-cli-templates), and [Ionic](https://github.com/pxblue/angular-cli-templates/blob/master/README.md#angular-cli-templates).

To start a new project with the CLI:

```sh
npx -p @pxblue/cli pxb new
```

For more detailed information and usage instructions, check out the [@pxblue/cli](https://www.npmjs.com/package/@pxblue/cli) package on NPM.

## Supported Frameworks

The PX Blue CLI works with all PX Blue supported frameworks: Angular, React, Ionic, and React Native. For React and React Native projects, you will have the option to create your project in TypeScript or JavaScript.
