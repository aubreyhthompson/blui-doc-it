## @brightlayer-ui/colors-branding

-   Incorporating BLUI colors into your project is handled differently depending on the framework that you are using.

Angular or Ionic

```
// in styles.scss or your top-level sass file
@import '~@brightlayer-ui/colors/palette.scss'
...
background-color: map-get($blui-blue, 500)
```

React or React Native

```
import * as Colors from '@brightlayer-ui/colors';
...
<div style={{background: Colors.blue['500']}}/>
```
